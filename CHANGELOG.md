# 변경 이력

이 프로젝트의 주요 변경사항을 기록합니다.

형식은 [Keep a Changelog](https://keepachangelog.com/ko/1.1.0/)를 따르며,
버전은 [Semantic Versioning](https://semver.org/lang/ko/)을 따릅니다.

## [0.4.14] - 베타

### 사건 일지 개편
- **필터 추가** — 중요도·인물·장소 드롭다운 + 전역 텍스트 검색 (제목·시점·장소·인물·내용)
- **드래그 앤 드롭 정렬** — 기존 "정렬 순서" 숫자 입력 제거, ⠿ 핸들로 직접 순서 변경
  - 드롭 시 `order` 값 10 간격으로 자동 재할당
  - 필터 적용 중엔 드래그 비활성화(보이는 순서와 실제 순서 혼동 방지)
- **작성 가이드 개선** — 빈 상태에 예시 문구, "시점" 필드에 더 풍부한 placeholder
- 삭제 확인에 사건 제목 표시

## [0.4.13] - 베타

### 추가됨
- **복선 ↔ 에디터 연결** — 본문에 `[[복선:이름]]` 태그를 심으면 복선 관리에서 자동 추적
  - 에디터에서 `[[복선:` 입력 시 자동완성 드롭다운 (기존 복선 검색 + 신규 생성)
  - ↑↓ 네비, Enter/Tab 삽입, Esc 닫기
  - 복선 카드에 `🔗 연결됨` 뱃지, 심기/회수 화 자동 계산
  - 심기 = 첫 등장 화, 회수 = 마지막 등장 화 (회수완료 상태일 때)
  - 중간 언급 화는 `💬 언급 N` 토글로 펼쳐보기
  - 화 라벨 클릭 시 해당 위치로 바로 점프
  - 복선 모달에 `태그 이름` 필드 추가 — 실시간으로 본문 연결 상태 표시
  - 태그 이름 변경 시 본문 태그 일괄 치환 확인 다이얼로그
- 일반용 확인 다이얼로그 `askDialog()` 추가 (삭제 전용 `confirmDialog`와 분리)

### 수정됨
- 복선 카드의 화 라벨이 "1화: 1화"처럼 중복 표시되던 문제 — 제목이 이미 번호형(`N화`, `N장`, `프롤로그` 등)이면 접두사 생략

### 호환성
- 기존 `뿌린 화/회수 화` 수동 입력 필드는 그대로 유지. 태그 연결이 있으면 자동 계산값 우선 표시, 없으면 수동값 fallback

## [0.4.12] - 베타

### 추가됨
- **전체 작품 검색** — 헤더 🔎 버튼 또는 `Ctrl+Shift+F`로 작품 전체를 한 번에 검색
  - 범위 선택: 본문 · 제목 · 회차 요약 · 메모 (독립 토글)
  - 대소문자 구분, 정규식 옵션
  - 화별로 그룹핑된 결과, 매칭 앞뒤 컨텍스트 표시
  - 결과 클릭 시 해당 화로 이동 + 정확한 위치 자동 선택
  - 요약/메모에 매칭이 있으면 메모 박스 자동 펼침
  - 범위·옵션 선택은 앱 디자인 톤과 맞춘 pill 토글 UI
- 집필 편집기 제목 input에 안정적 id(`chapter-title-input`) 부여

## [0.4.11] - 베타

### 추가됨
- 데이터 안전 패치: IndexedDB에 파일 핸들 저장, 페이지 숨김 시 저장 플러시, 15분 자동 스냅샷
- Undo/Redo 보강: 특수기호 삽입·찾아바꾸기에서 Ctrl+Z 지원, 화별 커스텀 스택
- 저장 상태 인디케이터 (헤더에 "저장됨 · N초 전")
- "모두 바꾸기" 확인 다이얼로그

### 변경됨
- 특수기호 삽입을 `execCommand('insertText')`로 전환하여 브라우저 undo 스택 유지
- 제목·상태 등 일부 필드를 실시간 저장으로 보강

### 수정됨
- `tryAutoReconnect`가 빈 함수였던 문제
- 탭 닫기 직전 debounce 타이머로 인한 최대 800ms 타이핑 유실

- 8색 컬러 테마 × 라이트/다크 모드
- PWA 지원
